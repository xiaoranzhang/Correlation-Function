# Smart Statistics for Social Sciences
# Author: Shawn_张

#作为R, SAS, SPSS 或者 MATLAB 的简化替代品。
长期以来数据分析处理软件都假定用户掌握了一定程度的统计学知识。所以，对很多科研工作者来说（特别是社会科学工作者），许多数据处理软件仍有一定的使用门槛。这个项目专为从事社会科学研究的人员而编写。


# Version Update
